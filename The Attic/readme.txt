Launching the Game
==================
If you're playing on Windows, the included batch file will start the game. On other systems, you can launch the game with the following command:

java engine.Starter attic.dat

Basic Commands
==============
For directions, both long and short version are possible:

north (n)
south (s)
east (e)
west (w)
northeast (ne)
northwest (nw)
southeast (se)
southwest (sw)
in (i)
out (o)
up (u)
down (d)

drop
open
close
look
inventory
